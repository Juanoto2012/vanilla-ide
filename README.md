# 🍦 Vanilla IDE

![Vanilla IDE Logo](https://file.garden/aKIWFnfXzhwEFeKS/vanilla%20ide/logo.png)

**Vanilla IDE** es un entorno de desarrollo ligero, rápido y sin distracciones, diseñado para desarrolladores que buscan volver a las bases sin sacrificar la potencia. Enfocado en la simplicidad y el rendimiento, es la herramienta ideal para proyectos que requieren un flujo de trabajo "puro".

---

## ✨ Características Principales

* **Zero-Config:** Empieza a programar al instante, sin perder horas en archivos de configuración complejos.
* **Ultraligero:** Consumo de recursos mínimo para que tu hardware se concentre en ejecutar tu código, no el editor.
* **Resaltado de Sintaxis Inteligente:** Soporte nativo para los lenguajes web modernos y de sistemas.
* **Terminal Integrada:** Acceso rápido a tus herramientas de consola sin salir de la ventana.
* **Extensibilidad Minimalista:** Añade solo lo que necesitas a través de un sistema de plugins ligero.

---

## 🚀 Instalación

Para comenzar a usar Vanilla IDE, clona este repositorio y sigue los pasos:

```bash
# Clonar el repositorio
git clone [https://github.com/Vanilla-IDE/vanilla-ide.git](https://github.com/Vanilla-IDE/vanilla-ide.git)

# Entrar al directorio
cd vanilla-ide

# Instalar dependencias (si aplica)
npm install

# Ejecutar la aplicación
npm start
```
